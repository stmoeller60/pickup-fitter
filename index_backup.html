Pickup Model Fitter

Loads 1 to 4 TXT files. The fit uses only the curves that are present. The additional load is never included in the fit itself, but is applied to the model afterwards only. New: Shape Focus blends between physical weighting and local peak/dip shape fidelity. In addition, a pure display offset is calculated for the H magnitude after fitting, so the shape remains unchanged and only the level is adjusted. New: The capacitance C is no longer searched freely over a wide range, but is automatically constrained based on the detected resonance frequency. Using the physical relation between resonance frequency, inductance, and capacitance, a plausible range for C is derived from the measured peak and the current L bounds. This improves fit stability, prevents non-physical solutions, and leads to more consistent and reproducible results, while keeping the overall model and fitting strategy unchanged. New: A known measurement load can now be included during the fit only. This allows reconstruction of the internal pickup model even when H and optionally Z were measured under a real interface input load such as Rin and Cin. The measurement load is removed completely after fitting. Any later RL/LL/CL load simulation always uses the pure identified internal pickup model only.

Measurement Load During Fit

Use known measurement load in fit
This known measurement load is applied only during the fit in order to reconstruct the internal pickup model from loaded measurements. After fitting, the measurement load is removed completely. The later RL / LL / CL simulation remains strictly separate and is always based on the pure internal model.

LCR Q Constraint

Use LCR Q constraint in fit
Optional soft constraint only. The Q term is added to the total fit error only when enabled and only for valid positive inputs. Q is evaluated from the full model input impedance at f_meas and therefore includes Rc automatically.

H Phase Correction

Apply -90° correction to H phase target
The correction is applied only to the target H phase used in the fit and plot. The model phase itself remains unchanged.
