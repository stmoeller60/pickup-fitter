Pickup Model Fitter

Loads 1 to 4 TXT files. The fit uses only the curves that are present. The additional load is never included in the fit itself, but is applied to the model afterwards only.
New: Shape Focus blends between physical weighting and local peak/dip shape fidelity. In addition, a pure display offset is calculated for the H magnitude after fitting, so the shape remains unchanged and only the level is adjusted.
New: The capacitance bounds are resonance-guided. The automatic resonance-based C bounds now apply only to C2, which is the main output shunt capacitance of the distributed model. The auxiliary capacitors C1, Cb1 and Cb2 are derived only via factors and are therefore not directly constrained by the resonance routine.
New: A known measurement load can now be included during the fit only. This allows reconstruction of the internal pickup model even when H and optionally Z were measured under a real interface input load such as Rin and Cin. The measurement load is removed completely after fitting. Any later RL/LL/CL load simulation always uses the pure identified internal pickup model only.

Measurement Load During Fit

Use known measurement load in fit
This known measurement load is applied only during the fit in order to reconstruct the internal pickup model from loaded measurements. After fitting, the measurement load is removed completely. The later RL / LL / CL simulation remains strictly separate and is always based on the pure internal model.

LCR Q Constraint

Use LCR Q constraint in fit
Optional soft constraint only. The Q term is added to the total fit error only when enabled and only for valid positive inputs. Q is currently evaluated as a series-equivalent Q from the full input impedance of the distributed model at f_meas: Q = abs(Im(Zin)) / Re(Zin).

H Phase Correction

Apply -90° correction to H phase target
The correction is applied only to the target H phase used in the fit and plot. The model phase itself remains unchanged.

Fit Strategy

0.45
First, the low-frequency level is stabilized. Then Shape Focus blends between the classical app error function and local shape fidelity around dip and peak.

Fit Control

Adaptive stop control for each start point. The fit continues while meaningful improvements occur, and stops when the target error is reached, the stall limit is exceeded, or the maximum iteration count per start is reached. Internally, the mutation strength is reduced in multiple refinement stages: coarse → medium → fine → very fine.
